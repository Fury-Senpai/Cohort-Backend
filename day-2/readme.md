## require vs import 
>  we use **require** or **import** to include modules in our application
**REQUIRE**
*  **require** is an old way to include modules in our web application and it is preffered in Node.Js as many framework and libraries are old
* **require** is synchornous , meaning it loads module at runtime.
**IMPORT**
* **import** was introduced in ES6. 
* **import** is a asynchronous allows static analysis
